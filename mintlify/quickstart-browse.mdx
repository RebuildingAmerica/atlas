## Overview

The Atlas directory is free to browse — no account needed. This quickstart walks through searching, filtering, and reading source attributions.

## Open the directory

Go to [atlas.rebuildingus.org/browse](https://atlas.rebuildingus.org/browse).

## Search by place

Enter a city or state in the search bar — for example, "Kansas City." Atlas returns entities tied to that location.

## Filter by issue area

Use the issue area filters to narrow results. Select **Housing and the Built Environment** to see only housing-related entities, or combine multiple filters to find cross-cutting work.

## Read an entry

Click any entry to see its full record:

- **Name and type** — Person, organization, initiative, campaign, or event
- **Description** — What they do, in 1–3 sentences
- **Location** — City, state, and geographic specificity
- **Issue areas** — The civic topics they connect to
- **Sources** — Every article, listing, or record where Atlas found this information

## Check the sources

This is the key differentiator. Every entry lists its sources with:
- Publication name
- Article title
- Publication date
- Link to the original

Click through to verify any entry independently.

## What you can do without an account

- Search and browse the full directory
- Filter by place, issue area, entity type, and source type
- Read full entry details and source attributions
- Flag incorrect entries (anonymous)
- Use the public API (100 requests/hour)

## What requires an account

- Save shortlists
- Run research (discovery runs)
- Export data
- Connect AI assistants via MCP
- Create API keys

See [Plans & pricing](/account/pricing) for details.
