# Connect Atlas to Gemini CLI

Google's Gemini CLI supports remote MCP servers with automatic OAuth discovery.

## Add the server

```bash
gemini mcp add atlas "https://atlas.rebuildingus.org/mcp" \
  --transport http \
  --scope user
```

Or edit `~/.gemini/settings.json` directly:

```json
{
  "mcpServers": {
    "atlas": {
      "url": "https://atlas.rebuildingus.org/mcp",
      "type": "http"
    }
  }
}
```

## Authorize

Gemini CLI discovers the OAuth server automatically via Atlas's protected resource metadata. On first use, it opens a browser for you to sign in and approve access. Tokens are persisted and refreshed automatically.

You can also trigger the OAuth flow manually:

```
/mcp auth atlas
```

## Requirements

<Warning>
MCP access requires **Atlas Pro** or **Atlas Team**. Free accounts cannot connect AI assistants. [See pricing](/account/pricing).
</Warning>

- Gemini CLI (latest version)
- Atlas Pro or Team plan
