# Connect Atlas to Claude Desktop

Claude Desktop supports remote MCP servers natively through Connectors. No config file needed — everything is set up through the UI.

## Setup

<Steps>

<Step title="Open Claude Desktop settings">
Open Claude Desktop and go to **Settings**.
</Step>

<Step title="Add a connector">
Select **Connectors** in the sidebar, then click **Add Connector**.
</Step>

<Step title="Enter the Atlas server URL">
```
https://atlas.rebuildingus.org/mcp
```
</Step>

<Step title="Authorize">
Click **Connect**. A browser window will open for you to sign in to Atlas and approve access.
</Step>

<Step title="Start using Atlas">
Once authorized, Atlas tools will appear in your conversations. Try asking Claude something like:

> Who's working on housing issues in Kansas City?
</Step>

</Steps>

## What you can do

With Atlas connected, you can ask Claude to:

- **Search for people and organizations** — "Find labor organizers in Detroit"
- **Explore a place** — "What civic work is happening in Boise, Idaho?"
- **Check coverage gaps** — "What issue areas are underrepresented in Kansas City?"
- **Get source details** — "Show me the sources for Prairie Workers Cooperative"
- **Understand demographics** — "What's the socioeconomic profile of Garden City, Kansas?"
- **Map issues to Atlas categories** — "What Atlas issue areas relate to affordable childcare?"

## Requirements

- Claude Desktop (latest version)
- Atlas Pro or Team plan — [see pricing](/account/pricing)
