# API quickstart

The Atlas API lets you search the civic research directory, explore places, and run discovery programmatically. This guide gets you from zero to a working API call in under 5 minutes.

## Base URL

All API requests go to:

```
https://atlas.rebuildingus.org/api
```

## Your first request

The simplest way to verify the API is reachable:

<CodeGroup>

```bash cURL
curl https://atlas.rebuildingus.org/health
```

```python Python
import httpx

response = httpx.get("https://atlas.rebuildingus.org/health")
print(response.json())
```

```typescript TypeScript
const response = await fetch("https://atlas.rebuildingus.org/health");
const data = await response.json();
console.log(data);
```

</CodeGroup>

## Search for entities

Find people and organizations working on an issue in a place. This endpoint is public — no authentication required.

<CodeGroup>

```bash cURL
curl "https://atlas.rebuildingus.org/api/places/kansas-city,missouri/entities?issue_area=housing"
```

```python Python
import httpx

response = httpx.get(
    "https://atlas.rebuildingus.org/api/places/kansas-city,missouri/entities",
    params={"issue_area": "housing"},
)

data = response.json()
for entity in data["data"]:
    print(f"{entity['name']} — {entity['description']}")
```

```typescript TypeScript
const params = new URLSearchParams({ issue_area: "housing" });

const response = await fetch(
  `https://atlas.rebuildingus.org/api/places/kansas-city,missouri/entities?${params}`
);

const { data } = await response.json();
data.forEach((entity) => {
  console.log(`${entity.name} — ${entity.description}`);
});
```

</CodeGroup>

## Get entity details

Retrieve full details for a specific entity, including all its sources:

<CodeGroup>

```bash cURL
curl https://atlas.rebuildingus.org/api/entities/{entity_id}
```

```python Python
entity = httpx.get(
    f"https://atlas.rebuildingus.org/api/entities/{entity_id}"
).json()

print(entity["data"]["name"])
for source in entity["data"].get("sources", []):
    print(f"  Source: {source['title']} ({source['publication']})")
```

</CodeGroup>

## Explore issue areas

Discover what issue areas Atlas tracks:

```bash
curl https://atlas.rebuildingus.org/api/issue-areas
```

Atlas organizes civic work into 11 domains and 47 issue areas. See the [taxonomy reference](/taxonomy) for the full listing.

## Check place coverage

See what Atlas knows about a place — and what's missing:

```bash
curl https://atlas.rebuildingus.org/api/places/kansas-city,missouri/coverage
```

The coverage endpoint returns entity counts by issue area and highlights gaps where Atlas has limited data.

## Authenticate for protected endpoints

To create entities, run discovery, or use workspace features, you need to authenticate. Atlas supports API keys and OAuth 2.1.

The quickest way to get started:

1. Sign up at [atlas.rebuildingus.org](https://atlas.rebuildingus.org)
2. Subscribe to [Atlas Pro](/account/pricing) ($5/month)
3. Go to **Account** > **API keys** > **Create key**
4. Add the key to your requests:

```bash
curl -H "X-API-Key: your-api-key" \
  https://atlas.rebuildingus.org/api/discovery-runs
```

See the full [authentication guide](/api-reference/authentication) for details on API keys, OAuth, and per-endpoint requirements.
