# Open source

Atlas is open source and designed to be independently useful beyond any single organization or initiative.

## License

Atlas is available on [GitHub](https://github.com/RebuildingAmerica/atlas). Check the repository for the current license terms.

## Relationship to Rebuilding America

Atlas is maintained by [The Rebuilding America Project](https://rebuildingus.org). It was created to support the broader mission of surfacing the people, organizations, and ideas rebuilding public life across America.

At the same time, Atlas is designed to stand on its own as a public resource. Its value is not limited to one organization or one media effort — it should be useful to anyone doing civic research, whether or not they've heard of Rebuilding America.

## Self-hosting

You can run your own Atlas instance. See the [self-hosting guide](/self-hosting/guide) for deployment instructions.

## Contributing

Atlas welcomes contributions — bug reports, feature suggestions, code, and documentation. See the [contributing guide](/resources/contributing).
