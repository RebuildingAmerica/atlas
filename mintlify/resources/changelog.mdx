# Changelog

<Steps>

<Step title="Documentation overhaul — April 22, 2026">
The public docs site has been restructured into a 4-tab, 40+ page resource organized by user intent:

- [Guides](/what-is-atlas) — Product overview, use cases, taxonomy, pricing, self-hosting
- [MCP & Agents](/mcp/overview) — Setup for 7 clients, tools reference, tutorials
- [API Reference](/api-reference/overview) — Auth, quickstart, rate limits, resource guides, endpoint reference
- [Resources](/resources/data-model) — Data model, glossary, FAQ, troubleshooting
</Step>

<Step title="Pricing and capability system — April 21, 2026">
Atlas now has a structured [capability system](/account/pricing) with three subscription tiers:

- **Free** — Browse, search, 2 research runs/month, public API at 100 req/hr
- **Atlas Pro** ($5/month) — Unlimited research, exports, 1 API key, MCP/OAuth
- **Atlas Team** ($25/month + $8/seat) — Shared workspace, unlimited API keys, watchlists, Slack, SSO
- **Research Pass** — Short-term Pro access ($9/30 days, $4/7 days)

Discounts available for journalists, nonprofits, and civic tech workers.
</Step>

<Step title="MCP server — April 2026">
Atlas exposes a remote [MCP server](/mcp/overview) for AI assistants with:

- OAuth 2.1 authentication with PKCE and dynamic client registration
- Streamable HTTP transport
- [10 tools](/mcp/tools) for entity search, place analysis, coverage gaps, and taxonomy resolution
- Setup guides for Claude Desktop, Claude Code, VS Code/Copilot, Gemini, Codex, and Copilot CLI
</Step>

<Step title="Enterprise SSO — April 2026">
Atlas Team workspaces now support enterprise single sign-on:

- Google Workspace OIDC
- Google Workspace SAML
- Domain-verified provider configuration
</Step>

</Steps>
