# Changelog

## April 2026

### Public documentation launch

Atlas documentation has been restructured into a complete, audience-driven site:
- **Guides** — Product overview, use cases, taxonomy, pricing, self-hosting
- **MCP & Agents** — Setup guides for Claude Desktop, Claude Code, VS Code/Copilot, Gemini CLI, Codex CLI, and Copilot CLI
- **API Reference** — Authentication, quickstart, rate limits, resource guides, and auto-generated endpoint reference
- **Resources** — Data model, glossary, FAQ, troubleshooting, and trust documentation

### MCP server

Atlas now exposes a remote MCP server at `https://atlas.rebuildingus.org/mcp` supporting OAuth 2.1 authentication and Streamable HTTP transport. 10 tools available for entity search, place analysis, coverage gaps, and taxonomy resolution.

### Pricing and capabilities

Atlas Pro ($5/month), Atlas Team ($25/month base + $8/seat), and Research Pass (from $4) are now available. Free tier includes browsing, searching, and 2 research runs per month.
