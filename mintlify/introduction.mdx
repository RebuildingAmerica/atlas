# Atlas Docs

Atlas is a source-linked civic research platform for places, entities, issue areas, and discovery workflows across the United States.

This Mintlify site is the new long-term home for Atlas documentation. The first migrated slice is the public API reference, generated from the repository's OpenAPI artifact at `openapi/atlas.openapi.json`.

## Start here

- Read the [migration guide](/migration) for how Mintlify fits alongside the existing `docs/` tree.
- Read the [Atlas overview](/product/overview) for product and platform context.
- Jump into the [API overview](/api-reference/overview) for the first public-facing Mintlify slice.

## Current migration stance

The existing Markdown docs in `docs/` remain in place while the Mintlify site grows into the primary Atlas docs surface.

That means:

- Mintlify owns the new API experience.
- The hand-written `docs/` tree remains the source for unmigrated material.
- The generated OpenAPI document remains the contract source of truth for endpoint reference content.
