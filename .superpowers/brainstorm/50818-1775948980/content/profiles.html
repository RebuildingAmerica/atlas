Key Differentiators

Person Profile

  • Dark header with name + role subtitle
  • Organization link — navigates to affiliated org
  • Source trail shows extraction context (the quotes about them)
  • Contact is compact (personal reach)
  • Issue focus is flat (what they personally work on)

Organization Profile

  • Dark header with name + scope/status badges
  • Key people — lists affiliated persons with roles
  • Issue footprint is grouped by domain (richer taxonomy view)
  • Contact is a grid (institutional presence)
  • Source trail is compact (type badges, no inline quotes)
